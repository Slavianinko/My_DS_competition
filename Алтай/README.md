## Задача:
Перед участниками чемпионата стояла задача предсказать будущий потенциальный статус студента на основе данных нескольких тысяч студентов.

## Итоги конкурса

Поставленная задача была решена с помощью ансамля моделей машинного обучения CatBoostClassifier, LightGBMClassifier и XGBoostClassifier. Качество обученной модели в моём решении обусловлено, тщательной предобработкой данных, сгенерированными дополнительными признаками и тонкой настройкой модели. Существенную прибавку качества модели удалось получить за счёт добавления сгенерированных признаков `Количество студентов в группе` и `Код_группы_2`. Судя по всему, количество студентов в группе действительно влияет на статус студента.
