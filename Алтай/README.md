## Задача:
Перед участниками чемпионата стояла задача предсказать будущий потенциальный статус студента на основе данных нескольких тысяч студентов.

## Данные:

1.   Код_группы
2.   Год_Поступления
3.   Пол            
4.   Основания        
5.   Изучаемый_Язык   
6.   Дата_Рождения    
7.   Уч_Заведение     
8.   Где_Находится_УЗ 
9.   Год_Окончания_УЗ 
10.  Пособие          
11.  Страна_ПП        
12.  Регион_ПП        
13.  Город_ПП         
14.  Общежитие         
15.  Наличие_Матери    
16.  Наличие_Отца      
17.  Страна_Родители   
18.  Опекунство        
19.  Село              
20.  Иностранец        
21.  КодФакультета     
22.  СрБаллАттестата   
23.  Статус

## Отработанные методы и навыки
- Исследовательский анализ данных;
- Создание признаков для обучения моделей;
- Работа с текстовыми данными;
- Построение моделей машинного обучения;
- Использование ансамблей моделей;
- Оценка важности признаков.
