## Задача:

Необходимо для каждого клиента из тестовой выборки предсказать target, который равен 1 если после коммуникации покупка совершена и если без коммуникации покупка не совершена, 0 в других случаях.

## Данные:

Участникам предоставляется несколько наборов данных для решения этой задачи.

Общие данные:
1. clients2.csv: информация о клиентах;
2. products.csv: информация о товарах;
3. train_purchases.csv: история покупок train клиентов;
4. test_purchases.csv: история покупок test клиентов.

Данные, относящиеся непосредственно к задаче:
1. train.csv: набор клиентов для обучения, с указанием treatment_flg — была ли совершена коммуникация, purchased — была ли совершена покупка; 
2. test.csv: список клиентов, для которых необходимо предсказать target.
